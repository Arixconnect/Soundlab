# Soundlab Studio

Premium onepage website voor Soundlab Studio, gericht op directe boekingen via Setmore.

## Aanpassen voor livegang

Vervang in `index.html` deze placeholders:

- `SETMORE_GENERAL_BOOKING_LINK`
- `SETMORE_PERSON_1_LINK`
- `SETMORE_PERSON_2_LINK`
- `SETMORE_PERSON_3_LINK`
- `WHATSAPP_LINK`
- `INSTAGRAM_LINK`

Vul daarna de echte namen, rollen, prijzen, locatie en eventuele studiofoto's in.
